## 常见问题：

A.不会再添加新组件、新功能，如果素材出现了错误（如素材名称标错、注释图标志错误等）都会及时勘误后更新的，若你在使用过程中发现了问题，欢迎私信联系我！

### Q.以后还会推出新模板吗？
A.会的，但是目前没有固定的更新频率，下次可能是一个月，也可能是半年后，这都无法确定，欢迎关注我的账号了解最新模板进度！

### Q.以后本模板会更新使用条约吗？
A.会的，可能会根据实际需求更新使用模板，可关注我的账号了解最新的使用条约！

### Q.可以提供模板的原文件吗？
A.很抱歉，不可以。

### Q.如果我希望本模板成为我的专属UI，可以买下它吗？
A.很抱歉，已公开的UI模板无法再买断，但你可以找我付费对此模板进行定制修改/拓展，使其成为全新的，作为买断使用的UI。

### Q.我在某平台上购买了本UI模板/我在某平台上看到了本UI模板正在售卖。
A.如果你购买了本UI模板，请立刻差评并退款！本UI模板为免费模板，不会在任何渠道上售卖！如果你是在某平台看到了本UI模板正在售卖，请立即举报，并私信告诉我，非常感谢！
